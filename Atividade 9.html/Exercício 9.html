Exercício 4: Por que usar o código 201?

O código 201 Created é usado porque ele é o padrão específico do protocolo HTTP para indicar que a requisição funcionou e que um novo recurso (como um cadastro) foi criado com sucesso no servidor.

Exercício 5: Passo a Passo do Cadastro de Usuário

  1. O usuário preenche os campos do formulário (nome, e-mail, senha) na página e clica em enviar.
  2. O JavaScript no front-end coleta esses dados e os formata em uma estrutura JSON.
  3. O navegador envia uma requisição HTTP do tipo POST contendo esse JSON para o servidor.
  4. O servidor processa a requisição, valida as informações e salva os dados no banco de dados.
  5. O servidor responde ao navegador com o status 201 Created confirmando o sucesso do cadastro.
