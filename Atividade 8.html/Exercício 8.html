Atividade 8: Como a web funciona

Exercício 1: Exemplos de GET

1. Acessar a página inicial do Google digitando o endereço no navegador.

2. Clicar no link de uma notícia para carregar o seu conteúdo de texto e imagens.

Exercício 2: Exemplos de POST

1. Preencher um formulário de cadastro com e-mail e senha e clicar em "Enviar".

2. Enviar uma nova mensagem em um chat ou publicar uma foto no Instagram.

Exercício 3: Código de Status 404

Significa que a comunicação com o servidor funcionou, mas a página ou o arquivo específico que você tentou acessar não existe ou o endereço digitado está incorreto (Não Encontrado).

Exercício 4: Código de Status 201

Indica que a requisição deu certo e um novo registro ou recurso foi criado com sucesso no banco de dados do servidor.

Exercício 5: Pedido e Resposta HTTP

Pedido (Request):

POST /avisos HTTP/1.1
Host: localhost:3000
Content-Type: application/json

{
  "titulo": "Feriado Escolar",
  "descricao": "Não haverá aula nesta sexta-feira."
}

Resposta (Response):

HTTP/1.1 201 Created
Content-Type: application/json

{
  "mensagem": "Aviso escolar cadastrado com sucesso!"
}
